Dúvidas

PERGUNTAS
FREQUENTES

Preciso saber programar para usar?
+
Não. O guia de instalação foi criado para quem nunca programou. Se você consegue seguir um tutorial do YouTube, consegue configurar essa automação. O Jarvis (IA de suporte) resolve qualquer dúvida técnica em segundos.
Qual o custo mensal para rodar a automação?
+
Menos de U$5/mês para a API do Claude. O n8n, Google Drive e Cloudinary são gratuitos para o volume de uso. O Shotstack tem plano gratuito para testes e plano pago (~U$49/mês) para produção em alta qualidade.
Para que tipo de canal funciona?
+
Esta automação é exclusiva para canais de COMPILAÇÃO DE ÁUDIO — música (qualquer gênero), lofi, pregações, frases motivacionais com voz, nature sounds, etc. NÃO funciona para vlogs, tutoriais, gameplay ou qualquer vídeo que precise de edição visual. Se seu canal posta vídeos com imagem em movimento, esta solução não é para você.
Ainda preciso criar as músicas manualmente?
+
Sim. A criação das músicas no Suno.com é a única etapa manual — e é onde entra seu toque criativo. Tudo depois disso (montar o vídeo, thumbnail, SEO, publicar) é 100% automático.
E se eu tiver problemas na configuração?
+
O Jarvis é um agente de IA treinado com 2 dias de construção real desta automação — incluindo todos os erros e soluções encontradas. Você digita o erro e ele resolve. Disponível 24 horas por dia.
